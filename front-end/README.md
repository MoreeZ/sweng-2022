## Download and run
#### Download
1. Clone the repository: `git clone https://github.com/MoreeZ/sweng-2022/` 
2. Navigate to local repository: `cd sweng-2022/front-end` 

#### Install nodejs packages
3. `npm install`

#### Run
4. Start the application: `npm start`
